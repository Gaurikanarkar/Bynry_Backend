# StockFlow – B2B Inventory Management System

## Structure

- **`part1/`** – Code review and debugging of the product creation endpoint
- **`part2/`** – Database schema design and analysis of missing requirements
- **`part3/`** – Low-stock alert API implementation using Node.js and Express

## Highlights

- **Part 1**: Resolved data consistency issues, enforced SKU uniqueness, and aligned the API with multi-warehouse business rules.
- **Part 2**: Designed a normalized and scalable database schema with clear constraints and identified key requirement gaps.
- **Part 3**: Implemented a production-ready low-stock alert API with proper error handling and well-defined assumptions.
